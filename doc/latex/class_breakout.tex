\doxysubsection{Detailed Description}
The main game class that controls every thing. 

In short, to run the game you should first create a instance of the class (i.\+e. break), then call break.\+config() to run the game.~\newline
 The main loop will run infinitely unless close the window, or press Q in the menu.~\newline
The program will log the situation of image texture \& audio resources loading and unloading, as well as the SDL library. All the resources should load and unload in a FIFO order, and must be in pair. It prints the error log to the screen if there occurs an incidence.
