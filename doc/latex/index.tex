This program is a \href{https://en.wikipedia.org/wiki/Breakout_(video_game)}{\texttt{ Breakout}} video game developed basically on \href{https://www.libsdl.org/}{\texttt{ SDL2 Library}}. There consists 3 levels of different difficulty, as well as providing two languages of English and Chinese for the player.\hypertarget{index_Features}{}\doxysection{Features}\label{index_Features}

\begin{DoxyItemize}
\item The game provides 3 levels that loaded from maps in the {\ttfamily map/} folder. The map is written in csv format, where different numbers represent bricks of various colors. Players can also customize the maps themselves.~\newline
 
\item Concerning the physics, the game engine is designed to implement a refraction angle after a collision, in order to make the movement harder to predicate. The ball would also accelerate whiling bouncing between the bricks.~\newline
  
\item The program provides a menu for entering certain instructions, which can be accessed by pressing {\ttfamily ESC}. Then the program halts to wait for a subsequent instruction. There are three instructions of quiting, restarting, and skipping current level. Press {\ttfamily ESC} to exit the menu and resume the game, and the game will continue instantly. ~\newline
  
\item There are various sound effects for different events in the game. Besides, a Jazz will infinitely play in the background. ~\newline
  
\item The icons, object textures and user interfaces are designed by myself, at the aim of providing a neat and beautiful panel. Resources from the website are all compliant to their own license, which is described in {\ttfamily license.\+md}. ~\newline
